W15

1.為何學習計算機程式需要建立個人倉儲?

需要上傳作業與學習如何維護倉儲。

2.全球資訊網能夠提供甚麼功能或有什麼優點?

是利用Internet的通訊能力，將文件、圖形、影像及聲音資料，透過簡單一致的界面，以及獨創的分散式超媒體資料查詢系統，讓使用者能夠輕鬆方便的，取得所要的資訊。

3.Replit, stud.cycu.org 與 localhost 分別代表甚麼?

三種維護網站的方法。

4.https, ssh 到底提供使用者那些功能或優點?

ssh 是一種加密的網路傳輸協定，可在不安全的網路中為網路服務提供安全的傳輸環境。

https是一種透過計算機網絡進行安全通信的傳輸協議，由權威數字證書機構驗證網站身份後頒發，可實現瀏覽器和網站服務器數據傳輸加密。

5.Brython 與 Python 有甚麼差別?

Brython 目的是可以用Python 代替Javascript 同样支持前端应用开发。

Python 用於Web 應用程式、軟體開發、資料科學與機器學習。

6.能夠直接在瀏覽器中以 Brython 繪製中華民國國旗, 或模擬綠色方塊在方塊格點中任意移動, 代表甚麼?

Brython很厲害。

7.變數, 字串, 整數, 浮點數, 數列, Tuple, Dictionary, 函式, 重複迴圈, class, 物件導向, 資料庫, 這些名詞對您而言代表甚麼?

可在程式裡利用這些指令，可以省去許多步驟。
